The CSS Background properties are used to add background effects in elements.

In this chapters, you will learn the following CSS background properties :

  • background-color
  • background-image
  • background-repat
  • background-attachment
  • background-position
  • background (shorthand property)

CSS Background-color

The background-color property specifies the background color of element.

Example

The background-color is set like this:

body {
background-color : lightblue;
}
